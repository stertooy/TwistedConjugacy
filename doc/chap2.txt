  [1X2.1 [33X[0;0YTwisted Conjugation Action[133X[101X
  
  [33X[0;0YLet  [23XG, H[123X be groups and [23X\varphi,\psi\colon H \to G[123X group homomorphisms. Then
  the pair [23X(\varphi,\psi)[123X induces a (right) group action on [23XG[123X given by[133X
  
  
  [24X[33X[0;6YG \times H \to G\colon (g,h) \mapsto g \cdot h := \psi(h)^{-1} g\varphi(h).[133X
  
  [124X
  
  [33X[0;0YThis  group action is called [13X[23X(\varphi,\psi)[123X-twisted conjugation[113X, and induces
  an  equivalence  relation  on  the  group  [23XG[123X. We say that [23Xg_1, g_2 \in G[123X are
  [23X(\varphi,\psi)[123X-twisted conjugate, denoted by [23Xg_1 \sim_{\varphi,\psi} g_2[123X, if
  and  only  if there exists some element [23Xh \in H[123X such that [23Xg_1 \cdot h = g_2[123X,
  or equivalently [23Xg_1 = \psi(h) g_2 \varphi(h)^{-1}[123X.[133X
  
  [33X[0;0YIf  [23X\varphi\colon  G  \to  G[123X  is  an  endomorphism  of  a  group  [23XG[123X, then by
  [13X[23X\varphi[123X-twisted   conjugacy[113X  we  mean  [23X(\varphi,\operatorname{id}_G)[123X-twisted
  conjugacy.  Most  functions in this package will allow you to input a single
  endomorphism  instead of a pair of homomorphisms. The "missing" endomorphism
  will  automatically  be  assumed to be the identity mapping. Similarly, if a
  single  group element is given instead of two, the second will be assumed to
  be the identity.[133X
  
  [1X2.1-1 TwistedConjugation[101X
  
  [33X[1;0Y[29X[2XTwistedConjugation[102X( [3Xhom1[103X[, [3Xhom2[103X] ) [32X function[133X
  
  [33X[0;0YImplements  the twisted conjugation (right) group action induced by the pair
  of homomorphisms ( [3Xhom1[103X, [3Xhom2[103X ) as a function.[133X
  
  [1X2.1-2 RepresentativeTwistedConjugation[101X

  [33X[0;0YTests whether the elements [3Xg1[103X and [3Xg2[103X are twisted conjugate under the twisted
  conjugacy action of the pair of homomorphisms ( [3Xhom1[103X, [3Xhom2[103X ).[133X
  
  [33X[0;0YThis  function  relies  on  the  output of [10XRepresentativeTwistedConjugation[110X.
  Computes an element that maps [3Xg1[103X to [3Xg2[103X under the twisted conjugacy action of
  the  pair of homomorphisms ( [3Xhom1[103X, [3Xhom2[103X ) or returns [9Xfail[109X if no such element
  exists.[133X

  4]. If [23XH[123X is finite, it relies on a stabiliser-orbit algorithm. Otherwise, it
  relies on a mixture of the algorithms described in [Rom16, Thm. 3], [BKL+20,
  Sec. 5.4], [Rom21, Sec. 7] and [DT21, Alg. 6].[133X

  [1X2.2 [33X[0;0YReidemeister Classes[133X[101X
  
  [33X[0;0YThe  equivalence classes of the equivalence relation [23X\sim_{\varphi,\psi}[123X are
  called    the    [13XReidemeister    classes    of    [23X(\varphi,\psi)[123X[113X    or   the
  [13X[23X(\varphi,\psi)[123X-twisted  conjugacy  classes[113X. We denote the Reidemeister class
  of  [23Xg  \in  G[123X  by  [23X[g]_{\varphi,\psi}[123X. The number of Reidemeister classes is
  called  the  Reidemeister  number  [23XR(\varphi,\psi)[123X  and is always a positive
  integer or infinity.[133X
  
  [1X2.2-1 ReidemeisterClass[101X
  
  [33X[1;0Y[29X[2XReidemeisterClass[102X( [3Xhom1[103X[, [3Xhom2[103X], [3Xg[103X ) [32X function[133X

  [33X[0;0YIf  [3Xhom1[103X  and [3Xhom2[103X are group homomorphisms from a group H to a group G, this
  method  creates  the  Reidemeister  class  of  the  pair  ([3Xhom1[103X,  [3Xhom2[103X) with
  representative [3Xg[103X. The following attributes and operations are available:[133X
  
  [30X    [33X[0;6Y[10XRepresentative[110X, which returns [3Xg[103X,[133X
  
  [30X    [33X[0;6Y[10XGroupHomomorphismsOfReidemeisterClass[110X,  which returns the list [ [3Xhom1[103X,
        [3Xhom2[103X ],[133X
  
  [30X    [33X[0;6Y[10XActingDomain[110X, which returns the group H,[133X
  
  [30X    [33X[0;6Y[10XFunctionAction[110X, which returns the twisted conjugacy action on G,[133X
  
  [30X    [33X[0;6Y[10XRandom[110X,  which  returns a random element belonging to the Reidemeister
        class,[133X
  
  [30X    [33X[0;6Y[10X\in[110X,  which  can  be  used  to  test  if  an  element  belongs  to the
        Reidemeister class,[133X
  
  [30X    [33X[0;6Y[10XList[110X,  which lists all elements in the Reidemeister class if there are
        finitely many, otherwise returns [9Xfail[109X,[133X
  
  [30X    [33X[0;6Y[10XSize[110X, which gives the number of elements in the Reidemeister class,[133X
  
  [30X    [33X[0;6Y[10XStabiliserOfExternalSet[110X,   which   gives   the   stabiliser   of   the

  [1X2.3 [33X[0;0YReidemeister Spectra[133X[101X
  
  [33X[0;0YThe  set  of  all  Reidemeister  numbers  of  automorphisms  is  called  the
  [13XReidemeister spectrum[113X and is denoted by [23X\operatorname{Spec}_R(G)[123X, i.e.[133X
  
  
  [24X[33X[0;6Y\operatorname{Spec}_R(G)    :=    \{\,    R(\varphi)    \mid   \varphi   \in
  \operatorname{Aut}(G) \,\}.[133X
  
  [124X
  
  [33X[0;0YThe  set of all Reidemeister numbers of endomorphisms is called the [13Xextended
  Reidemeister spectrum[113X and is denoted by [23X\operatorname{ESpec}_R(G)[123X, i.e.[133X
  
  
  [24X[33X[0;6Y\operatorname{ESpec}_R(G)    :=    \{\,    R(\varphi)   \mid   \varphi   \in
  \operatorname{End}(G) \,\}.[133X
  
  [124X
  
  [33X[0;0YThe set of all Reidemeister numbers of pairs of homomorphisms from a group [23XH[123X
  to  a group [23XG[123X is called the [13Xcoincidence Reidemeister spectrum[113X of [23XH[123X and [23XG[123X and
  is denoted by [23X\operatorname{CSpec}_R(H,G)[123X, i.e.[133X
  
  
  [24X[33X[0;6Y\operatorname{CSpec}_R(H,G)  :=  \{\, R(\varphi, \psi) \mid \varphi,\psi \in
  \operatorname{Hom}(H,G) \,\}.[133X
  
  [124X
  
  [33X[0;0YIf  [3XH[103X  = [3XG[103X this is also denoted by [23X\operatorname{CSpec}_R(G)[123X. The set of all
  Reidemeister numbers of pairs of homomorphisms from every group [23XH[123X to a group
  [23XG[123X   is   called   the   [13Xtotal   Reidemeister  spectrum[113X  and  is  denoted  by
  [23X\operatorname{TSpec}_R(G)[123X, i.e.[133X
  
  
  [24X[33X[0;6Y\operatorname{TSpec}_R(G) := \bigcup_{H} \operatorname{CSpec}_R(H,G).[133X
  
  [124X
  
  [33X[0;0YPlease note that the functions below are only implemented for finite groups.[133X

  [1X2.4 [33X[0;0YReidemeister Zeta Functions[133X[101X
  
  [33X[0;0YLet    [23X\varphi,\psi\colon    G    \to   G[123X   be   endomorphisms   such   that
  [23XR(\varphi^n,\psi^n) < \infty[123X for all [23Xn \in \mathbb{N}[123X. Then the [13XReidemeister
  zeta function[113X [23XZ_{\varphi,\psi}(s)[123X of the pair [23X(\varphi,\psi)[123X is defined as[133X
  
  
  [24X[33X[0;6YZ_{\varphi,\psi}(s)  := \exp \sum_{n=1}^\infty \frac{R(\varphi^n,\psi^n)}{n}
  s^n.[133X
  
  [124X
  
  [33X[0;0YPlease  note that the functions below are only implemented for endomorphisms
  of finite groups.[133X
  
  [1X2.4-1 ReidemeisterZetaCoefficients[101X
  
  [33X[1;0Y[29X[2XReidemeisterZetaCoefficients[102X( [3Xendo1[103X[, [3Xendo2[103X] ) [32X function[133X
  
  [33X[0;0YFor  a finite group, the sequence of Reidemeister numbers of the iterates of
  [3Xendo1[103X  and [3Xendo2[103X, i.e. the sequence [23XR([3Xendo1[103X,[3Xendo2[103X)[123X, [23XR([3Xendo1[103X^2,[3Xendo2[103X^2)[123X, ...,
  is  eventually  periodic,  i.e. there exist a periodic sequence [23X(P_n)_{n \in
  \mathbb{N}}[123X  and  an  eventually zero sequence [23X(Q_n)_{n \in \mathbb{N}}[123X such
  that[133X
  
  
  [24X[33X[0;6Y\forall n \in \mathbb{N}: R(\varphi^n,\psi^n) = P_n + Q_n.[133X
  
  [124X
  
  [33X[0;0YThis  function  returns  a  list  containing two sublists: the first sublist
  contains  one  period  of  the sequence [23X(P_n)_{n \in \mathbb{N}}[123X, the second
  sublist  contains  [23X(Q_n)_{n  \in \mathbb{N}}[123X up to the part where it becomes
  the constant zero sequence.[133X
